PakThreads

PakThreads is a modern online discussion platform tailored for Pakistani audiences, enabling users to create posts, engage in discussions, and explore trending topics in a Reddit-style interface. Built with Angular and .NET, it supports dynamic post types, tag filtering, and a clean user experience with a powerful backend.
